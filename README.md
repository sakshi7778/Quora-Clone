# 📌 Quora Clone (Node.js + Express + EJS)

## 📖 About
This is a simple Quora-like web application where users can create, read, update, and delete posts.  
This project mainly focuses on backend functionality.

## 🛠️ Tech Stack
- Node.js  
- Express.js  
- EJS  
- UUID  

## ✨ Features
- Create Post  
- View All Posts  
- Edit Post  
- Delete Post  

## ▶️ How to Run
1. Clone the repository  
2. Install dependencies  
   
   npm install
   
3. Run the server  
   
   node index.js
   
4. Open in browser  
   
   http://localhost:8080/posts
   

## ⚠️ Note
This project focuses more on functionality than UI design.
